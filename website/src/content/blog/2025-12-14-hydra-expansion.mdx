# Hydra: расширенные возможности

Мы значительно расширили интеграцию с [Hydra](https://hydra.ojack.xyz/) — live-кодируемым видеосинтезатором от Olivia Jack.

## Улучшенная инициализация

Теперь `initHydra()` работает корректно и стабильно:

```javascript
await initHydra()
osc(10, 0.1, 0.8)
  .rotate(() => time * 0.1)
  .kaleid(4)
  .out()
```

Инициализация:
- **Асинхронная** — используйте `await initHydra()` в начале кода
- **Автоматическая настройка canvas** — визуал отображается поверх редактора
- **Интеграция с аудио** — визуализация реагирует на звук через `a.fft`

## Аудиореактивность

Особенно полезна связка Hydra + Strudel для аудиовизуальных перформансов:

```javascript
await initHydra()

// Визуал реагирует на бас
osc(() => a.fft[0] * 60, 0.1, 0.8)
  .kaleid(() => Math.floor(a.fft[1] * 8) + 3)
  .out()

$: s("bd sd [bd bd] sd")
```

Массив `a.fft[]` содержит частотный спектр аудио — от низких (бас) к высоким частотам.
