# Hydra - Полное руководство на русском языке

Добро пожаловать в полную документацию по **Hydra** - браузерному live-coding видеосинтезатору для создания визуальных эффектов в реальном времени.

## Что такое Hydra?

**Hydra** - это бесплатный инструмент с открытым исходным кодом для live-coding сетевых визуализаций, вдохновлённый аналоговыми модульными синтезаторами. Создан художницей [Olivia Jack](https://ojack.xyz/).

### Основные возможности

- Работает прямо в браузере (включая мобильные устройства)
- Написан на JavaScript, компилируется в WebGL
- Синтаксис вдохновлён аналоговым модульным синтезом
- Поддержка веб-камеры, видео, изображений
- Аудио-реактивные визуализации через FFT
- Сетевое взаимодействие между пользователями через WebRTC
- Интеграция с p5.js, Three.js, Tone.js и другими библиотеками

## Быстрый пример

```javascript
// Простой осциллятор с калейдоскопом
osc(10, 0.1, 0.8)
  .kaleid(4)
  .rotate(0.5)
  .out()
```

## Использование в Strudel

Hydra интегрирована в Strudel! Используйте `await initHydra()` в начале кода для активации визуализаций. Подробнее в разделе [Hydra в Strudel](../learn/21-hydra.md).
