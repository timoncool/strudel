# Sounds

Мы можем воспроизводить звуки с помощью `s` двумя разными способами:

- `s` может запускать audio samples, где звуковой файл загружается в фоновом режиме и воспроизводится:
  
```javascript
// Пример:
s("bd hh sd hh")
```

- `s` может запускать audio synthesisers, которые синтезируются в реальном времени с помощью кода, также работающего в фоновом режиме:
  
```javascript
// Пример:
s("sawtooth square triangle sine")
```

Вы можете узнать больше об обоих этих подходах на страницах [Synths](/learn/synths) и [Samples](/learn/samples).

# Комбинирование нот и звуков

В обоих вышеприведенных случаях мы больше не контролируем напрямую `note`/`freq` звука, слышимого через `s`, как это было на странице [Notes](/workshop/first-notes/).

Так как же мы можем контролировать и звук, и высоту тона? Мы можем _комбинировать_ `note`/`freq` с `s`, чтобы изменить звук наших высот:

```javascript
// Пример:
note("a3 c#4 e4 a4").s("sawtooth")
```

```javascript
// Пример:
note("57 61 64 69").s("sine")
```

```javascript
// Пример:
freq("220 275 330 440").s("triangle")
```

Последний пример на самом деле будет звучать одинаково с `s` или без него, потому что `triangle` — это значение по умолчанию для `s`.

Что насчет комбинирования разных нот с разными звуками одновременно?

```javascript
// Пример:
freq("220 275 330 440 550").s("triangle sawtooth sine")
```

Хм, здесь происходит что-то интересное, связанное с тем, что есть пять нот и три звука.
