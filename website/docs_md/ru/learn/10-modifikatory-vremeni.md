# Модификаторы времени

Следующие функции каким-либо образом изменяют временную структуру pattern.
Некоторые из них имеют эквивалентные операторы в Mini Notation:

| функция                | mini         |
| ---------------------- | ------------ |
| `"x".slow(2)`          | `"x/2"`      |
| `"x".fast(2)`          | `"x*2"`      |
| `"x".euclid(3,8)`      | `"x(3,8)"`   |
| `"x".euclidRot(3,8,1)` | `"x(3,8,1)"` |
