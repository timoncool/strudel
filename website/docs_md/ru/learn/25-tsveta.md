# Цвета

Strudel поддерживает работу с цветами для визуализации паттернов. Вы можете использовать различные цветовые форматы:

- Названия цветов: `"red"`, `"blue"`, `"cyan"`, `"magenta"` и т.д.
- HEX коды: `"#FF0000"`, `"#00FF00"`, `"#0000FF"`
- RGB значения: `rgb(255, 0, 0)`, `rgb(0, 255, 0)`

## Использование цветов

Цвета можно применять к паттернам с помощью функции `.color()`:

```javascript
// Пример:
note("c a f e").color("cyan magenta")
```

## Цвета для визуализации

Цвета особенно полезны при использовании визуализаций, таких как pianoroll, punchcard и других:

```javascript
// Пример:
note("c3 e3 g3 b3")
  .color("red green blue yellow")
  .pianoroll()
```

## Паттерны цветов

Вы можете создавать паттерны цветов, которые будут меняться со временем:

```javascript
// Пример:
note("c d e f g a b")
  .color("<red blue green yellow>")
  .scope()
```
