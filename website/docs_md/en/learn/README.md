## Conversion Rules

When converting from MDX to Markdown, the following rules were applied:

1. Removed component imports (import ... from)
2. JSX components (MiniRepl, JsDoc, etc.) replaced with plain Markdown
3. MiniRepl components converted to ```javascript code blocks or placeholders
4. JsDoc components replaced with function headers or placeholders
5. All code examples preserved in ```javascript blocks
6. Frontmatter preserved with title
7. Heading structure maintained
8. Files numbered for ordered reading
