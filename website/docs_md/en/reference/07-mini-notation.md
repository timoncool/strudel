# Мини-нотация

Мини-нотация - это специальный синтаксис для записи паттернов в Strudel.

## Базовый синтаксис

- Пробелы разделяют события в последовательности
- `[]` - группировка событий
- `<>` - альтернативные события (чередование)
- `*` - повторение
- `/` - замедление
- `,` - наслоение (играть одновременно)
- `!` - репликация
- `?` - случайное присутствие
- `_` - пауза/тишина
- `~` - легато
- `@` - удлинение

## Примеры

```javascript
// Простая последовательность
note("c d e f")

// Группировка
note("c [d e] f")

// Альтернативы
note("<c e g> <d f a>")

// Повторение
note("c*3 d")

// Замедление
note("c/2 d e")

// Наслоение
note("[c,e,g] [d,f,a]")

// Случайность
note("c? d e?")
```
