# Устройства ввода

Strudel поддерживает различные устройства ввода, такие как геймпады и MIDI контроллеры, для манипулирования patterns в реальном времени.

## Геймпад

Вы можете использовать геймпад (например, Xbox или PlayStation контроллер) для управления паттернами:

```javascript
// Пример:
// Получить первый подключенный геймпад
const gp = gamepad(0)

// Использовать кнопки
note("c3 e3 g3 b3")
  .mask(gp.a)  // Играет только когда нажата кнопка A
  .s("piano")

// Использовать стики
note("c3 e3 g3")
  .lpf(gp.lstick.x.range(200, 5000))  // Левый стик контролирует фильтр
  .gain(gp.lstick.y.range(0, 1))      // Вертикаль контролирует громкость
  .s("sawtooth")
```

## Доступные элементы управления геймпада

### Кнопки:
- `gp.a`, `gp.b`, `gp.x`, `gp.y` - основные кнопки
- `gp.lb`, `gp.rb` - бамперы (L1/R1)
- `gp.lt`, `gp.rt` - триггеры (L2/R2)
- `gp.start`, `gp.select` - системные кнопки

### Стики:
- `gp.lstick.x`, `gp.lstick.y` - левый стик (X и Y оси)
- `gp.rstick.x`, `gp.rstick.y` - правый стик (X и Y оси)

### D-Pad:
- `gp.up`, `gp.down`, `gp.left`, `gp.right` - крестовина

## MIDI контроллеры

Для MIDI устройств смотрите раздел [MIDI, OSC и MQTT](/learn/input-output).

## Клавиатура и мышь

Также доступны:
- `mousex`, `mousey` - позиция мыши
- События клавиатуры (в разработке)

```javascript
// Пример использования мыши:
note("c3 e3 g3")
  .lpf(mousex.range(200, 5000))  // Горизонталь контролирует фильтр
  .gain(mousey.range(0, 1))      // Вертикаль контролирует громкость
  .s("sawtooth")
```
