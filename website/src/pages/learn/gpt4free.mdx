# GPT4Free — бесплатный AI-агент

GPT4Free — это open-source проект, который предоставляет бесплатный доступ к различным языковым моделям. Bulka интегрирует GPT4Free, чтобы AI-агент был доступен каждому — без API ключей и подписок.

## Быстрый старт

1. Открой вкладку **Chat** в боковой панели справа
2. Нажми на иконку настроек (шестерёнка)
3. Выбери провайдера **GPT4Free**
4. Выбери модель из списка (например, GPT-4o или Claude)
5. Выбери провайдера из списка
6. Начинай общаться с агентом!

API ключ не требуется.

## Доступные модели

GPT4Free предоставляет доступ к различным моделям:

- **GPT-4o** и **GPT-4o Mini** — модели OpenAI
- **Claude 3.5 Sonnet** — модель Anthropic
- **Gemini** — модели Google
- И многие другие

Список моделей и провайдеров загружается автоматически.

## Важные ограничения

### Инструменты не работают

Модели через GPT4Free **не поддерживают function calling** (tools). Это значит что агент:

- ❌ Не может автоматически запускать и останавливать музыку
- ❌ Не может искать по документации Strudel
- ❌ Не может смотреть примеры кода
- ❌ Не может выделять фрагменты в редакторе

Агент будет генерировать код, но тебе нужно самостоятельно копировать его в редактор и запускать.

### Возможные проблемы

- **Реклама** — некоторые провайдеры могут вставлять рекламные сообщения
- **Лимиты** — количество запросов ограничено
- **Нестабильность** — провайдеры могут периодически не работать
- **Медленные ответы** — скорость зависит от загрузки провайдера

### Что работает

- ✅ Генерация кода для музыки
- ✅ Объяснение как работает код
- ✅ Помощь с синтаксисом Strudel
- ✅ Ответы на вопросы о лайв-кодинге

## Советы по использованию

### Подбирайте провайдеров

Если один провайдер не отвечает или выдаёт ошибки — попробуйте другой. Разные провайдеры имеют разную доступность.

### Копируйте код вручную

Поскольку инструменты не работают, копируй сгенерированный код из чата в редактор вручную (Ctrl+C / Ctrl+V).

### Будьте терпеливы

Бесплатные провайдеры могут отвечать медленнее платных API.

## Для максимального опыта

Если хочешь полноценного AI-агента со всеми инструментами:

1. Получи API ключ от [Anthropic](https://console.anthropic.com/)
2. В настройках чата выбери провайдера **Anthropic**
3. Введи API ключ
4. Выбери модель **Claude Opus 4.5** или **Claude Sonnet 4.5**

В этом режиме агент:
- Ищет по документации и находит нужные функции
- Смотрит примеры использования
- Автоматически редактирует код в редакторе
- Запускает и останавливает музыку
- Выделяет фрагменты кода
