# Движение устройства

Strudel поддерживает использование сенсоров движения устройства (акселерометр, гироскоп) для управления паттернами в реальном времени.

## Использование датчиков движения

На мобильных устройствах вы можете использовать данные акселерометра и гироскопа для манипулирования звуком:

```javascript
// Пример:
// Акселерометр
note("c3 e3 g3")
  .lpf(accel.x.range(200, 2000))  // Фильтр зависит от наклона
  .s("sawtooth")

// Гироскоп
note("c3 e3 g3")
  .gain(gyro.y.range(0, 1))  // Громкость зависит от вращения
  .s("sine")
```

## Доступные датчики

- `accel.x` - ускорение по оси X
- `accel.y` - ускорение по оси Y
- `accel.z` - ускорение по оси Z
- `gyro.x` - вращение по оси X
- `gyro.y` - вращение по оси Y
- `gyro.z` - вращение по оси Z

## Практические примеры

Встряхивание устройства для управления дисторшеном:

```javascript
// Пример:
note("c2 e2 g2")
  .s("sawtooth")
  .distort(accel.z.range(0, 2))  // Встряхивание добавляет дисторшен
  .lpf(800)
```
