# Сигналы

# Непрерывные сигналы

Сигналы - это patterns с непрерывными значениями, что означает, что они имеют теоретически бесконечное количество шагов.
Они могут предоставлять потоки чисел, которые можно сэмплировать в дискретные моменты времени.
