# Notes

Высота тона — важный строительный блок во многих музыкальных традициях.
В Strudel высота тона может быть выражена в виде имен нот, номеров нот или частот.
Вот один и тот же pattern, записанный тремя разными способами:

- `note`: буквенная нотация, подходит для тех, кто знаком с западной теорией музыки:

```javascript
// Пример:
note("a3 c#4 e4 a4")
```

- `note`: числовая нотация, подходит для тех, кто хочет использовать узнаваемые высоты, но не заботится о теории музыки:

```javascript
// Пример:
note("57 61 64 69")
```

- `freq`: нотация частоты, подходит для тех, кто хочет выйти за рамки стандартизированных систем настройки:

```javascript
// Пример:
freq("220 275 330 440")
```

Давайте рассмотрим их более подробно...

## Имена `note`

Имена нот могут быть записаны с буквой ноты, за которой следует номер октавы. Вы можете обозначить бемоли с помощью `b`, а диезы с помощью `#`.

```javascript
// Пример:
note("a3 c#4 e4 a4")
```

Кстати, вы можете редактировать содержимое плеера и нажимать "update", чтобы услышать ваше изменение!
Вы также можете нажать "play" на следующем плеере без необходимости останавливать последний.

## Числа `note`

Если хотите, вы также можете использовать числа с `note`:

```javascript
// Пример:
note("57 61 64 69")
```

Эти числа интерпретируются как так называемые [MIDI numbers](https://www.inspiredacoustics.com/en/MIDI_note_numbers_and_center_frequencies), где соседние целые числа отстоят друг от друга на один 'semitone'.

Вы также можете писать десятичные числа, чтобы получить 'microtonal' высоты (между черными и белыми клавишами piano):

```javascript
// Пример:
note("74.5 75 75.5 76")
```

## `freq`

Чтобы получить максимальную свободу, вы также можете использовать `freq` для прямого управления частотой:

```javascript
// Пример:
freq("220 275 330 440")
```

## Слух и частота

В приведенном выше примере мы воспроизводим A3 (220Hz), C#4 natural (275Hz), E4 (330Hz) и A4 (440Hz), отражая наши предыдущие примеры.

Но можете ли вы услышать разницу между этими отдельными частотами?

```javascript
// Пример:
freq("220 221 223 224")
```

А эти?

```javascript
// Пример:
freq("2020 2021 2023 2024")
```

Чем выше мы поднимаемся...

```javascript
// Пример:
freq("5020 5021 5023 5024")
```

Тем меньше расстояния мы можем услышать между частотами!

```javascript
// Пример:
freq("10020 10021 10023 10024")
```

Почему это так? [Человеческий слух работает логарифмически](https://www.audiocheck.net/soundtests_nonlinear.php).

## От нот к звукам

На этой странице, когда мы воспроизводили pattern нот вот так:

```javascript
// Пример:
note("a3 c#4 e4 a4")
```

Мы слышали простой синтезированный звук, на самом деле мы слышали [triangle wave oscillator](https://en.wikipedia.org/wiki/Triangle_wave).

Это синтезатор по умолчанию, используемый Strudel, но как же тогда создавать разные звуки в Strudel?

Давайте узнаем об этом на следующей странице [Sounds](/learn/sounds).
